# `docs/` — Documentación y entregables del proyecto

Contiene los entregables escritos del proyecto de Minería de Datos (CC442) y la
documentación interna del equipo.

## Contenido

| Ruta | Qué es |
|---|---|
| `paper/` | Artículo científico en formato IEEE (fuente LaTeX + PDF compilado). Entregable 1. |
| `presentation/` | Presentación de la defensa en Beamer (fuente LaTeX + PDF, 22 diapositivas). |
| `bracket.html` | Demo web del simulador del Mundial 2026 (predicción out-of-distribution). |
